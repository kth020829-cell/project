**1. 데이터 전처리 및 파이프라인 (Data Engineering)**

- **불필요 변수 제거**: 예측과 무관한 `ID` 컬럼을 제거하여 모델 노이즈 최소화.
- **변수 타입 체계화**: 수치형 변수와 Binary Feature를 명확히 구분하고, 이진 변수 매핑을 통해 입력 데이터 일관성 확보.
- **스케일링 & 인코딩**: 모델별 특성에 맞춰 스케일링과 인코딩을 적용하여 학습 안정성 향상.
- **EDA 기반 검증**: Binary / Non-binary 변수 분포 시각화를 통해 데이터 편향 및 이상치 사전 점검.

**2. 모델링 및 예측 전략 (ML Engineering)**

- **서브미션 앙상블 전략**: 서로 다른 전처리·모델링 전략을 적용한 3개 모델 결과를 앙상블하여 단일 모델 대비 일반화 성능 강화.
- **모델별 독립 최적화**: 각 모델에 대해 개별 전처리 및 하이퍼파라미터 튜닝을 수행하여 성능 극대화.
- **임계값(Threshold) 최적화**: Cross-Validation 기반으로 Optimal Threshold를 도출하고, 단순 0.5 기준이 아닌 데이터 분포에 맞춘 예측 전략 적용.
- **클래스 불균형 고려**: Positive class 분포 분석을 통해 평가 지표 중심의 의사결정 수행.

역할: 데이터 스케일링 및 인코딩, 앙상블 전략 도출
