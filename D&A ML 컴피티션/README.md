**프로젝트명: 미국 거주자 건강 데이터를 활용한 심장마비 예측**

**1. 데이터 전처리 및 파이프라인**

- **자동화 파이프라인:** `Pipeline`과 `ColumnTransformer`를 활용해 결측치 대체(평균/최빈값), 스케일링(MinMax), 인코딩 과정을 체계화.
- **인코딩 최적화:** '차원의 저주' 방지를 위해 `BinaryLabelTransformer` 커스텀 클래스를 구현하여 이진 변수 처리 효율성 극대화.
- **다중공선성 제거:** 상관관계 분석을 통해 유사 피처(Weight, Obesity 등)를 제거하고 `BMI`로 단일화하여 모델 안정성 확보.

**2. 피처 엔지니어링 및 모델링**

- **도메인 특화 피처:** 비만도(Obesity), 수면 편차, 예방 관리 점수 등 5종의 신규 파생 변수를 생성하여 예측력 강화.
- **불균형 데이터 대응:** 타겟 클래스 불균형 해소를 위해 **SMOTE** 오버샘플링 기법 적용.
- **앙상블 모델 최적화:** **Logistic Regression**과 **LightGBM**을 결합한 **Soft Voting** 앙상블 모델을 구축하고 `GridSearchCV`로 하이퍼파라미터 최적화.(ROC-AUC 0.867 달성)

역할: 데이터 전처리, 앙상블 모델 최적화
