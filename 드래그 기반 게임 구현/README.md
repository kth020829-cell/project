**Pygame 라이브러리를 활용한 클래스 기반 2D 생존형 액션 게임 개발**

**1. 객체 지향 기반 게임 시스템 설계** 

- **모듈화 설계:** 게임의 유지보수와 확장성을 고려하여 `Main`(실행), `Player`(캐릭터), `Villain`(적), `Item`(아이템) 등 핵심 기능을 클래스 단위로 분리하여 구조적으로 설계.
- **상태 관리 및 UI 구현:** 생존 시간, 처치 수(Killed) 등 실시간 데이터를 추적하는 UI 시스템과 보스 등장 및 'Fever Time' 이벤트 로직을 구현.

**2. 인터랙티브 알고리즘 및 물리 구현**

- **직관적 조작 체계:** 기존 방향키 방식에서 벗어나 마우스 드래그 및 터치 좌표를 실시간으로 추적하여 캐릭터가 이동하는 직관적인 컨트롤 시스템 구현.
- **지능형 적 추적 로직:** 적(Villain)이 플레이어를 추적할 때 단순 이동이 아닌 '랜덤 오프셋'과 '추적 비율'을 결합하여 예측 불가능한 움직임을 구현.
- **충돌 판정 및 투사체 로직:** 아이템 획득 시 여러 방향(벡터)으로 발사되는 투사체(Bone) 알고리즘과 객체 간 충돌 판정 시스템 구축.

**3. 문제 해결 및 고도화** 

- **멀티 채널 오디오 제어:** 보스 등장 등 게임 상황 변화에 따른 배경음악(BGM)의 깔끔한 전환을 위해 `pygame.mixer` 모듈의 리소스 관리 문제를 해결.
- **성장형 게임 밸런스:** 단순 회피 게임을 넘어 아이템 조합과 스킬 강화를 통해 캐릭터가 성장하는 '로그라이크' 요소를 가미하여 게임성 강화.
